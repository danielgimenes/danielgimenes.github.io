The Fowler Model

[Fowler PoEAA]

  • Presentation: responsible for defining and managing user interfaces in displaying data and accepting inputs to the application;
  • Domain / Business: responsible for representing business elements behaviour and data, rules and interaction between themselves;
  • Data Access / Data Source: responsible for storing and retrieving data from a data source (database, message queue, etc);
